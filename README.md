# Player-JS

A simple music player built for JavaScript study purposes. This project focuses on DOM manipulation and basic logic in Vanilla JS.

## 🚀 Features
- **Play / Pause**: Toggle music playback.
- **Navigation**: Go to the next or previous song in the playlist.
- **Dynamic Content**: Updates song name, artist, and album art automatically.
- **Dark Theme**: Clean UI with a modern dark aesthetic.

## 🛠️ Technologies
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript
- [Bootstrap Icons](https://icons.getbootstrap.com/)

## 📖 How to run
1. Clone or download this repository.
2. Open `index.html` in your web browser.
